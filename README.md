# @tractorcow/beefdiff

A command-line tool for beefdiff.

## Installation

```bash
npm install -g @tractorcow/beefdiff
```

Or use with `npx`:

```bash
npx @tractorcow/beefdiff
```

## Usage

```bash
beefdiff
```

## Development

### Prerequisites

- Node.js 22 or higher
- npm

### Setup

```bash
npm install
```

### Build

```bash
npm run build
```

### Clean

```bash
npm run clean
```
